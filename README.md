<b>Local deployment</b> :
1. Go to server directory and install dependencies using command
```bash
yarn
```
2. Run server
```bash
yarn start
```
3. Open second console, go to client directory and install dependencies using command
```bash
yarn
```
4. Run client
```bash
yarn start
```
5. Open project at localhost
